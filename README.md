## 🌟 Features

*   **100% Accurate Scale Calibration:** Calibrate dynamically with a physical object (credit card, driving license, or coin) or look up your specific device model automatically.
*   **Multiple Units & Modes:** Toggle effortlessly between **cm ruler online**, **inch ruler online**, millimeters (mm), and screen pixels (px).
*   **No Installation Required:** A lightweight, offline-first **ruler app browser** experience that launches in under a second.
*   **Full-Screen Edge Ruler Overlay:** An immersive interactive mode lining up rulers on all four display edges with draggable measuring guides.
*   **Responsive Fluid Interface:** Full support for both desktop monitors and mobile touch screens, styled with a modern aesthetic including an eye-safe Dark Mode.

## 🛠️ Interactive Measuring Tools

Beyond the standard **screen ruler**, RuleScale provides a robust suite of virtual tools:

1.  **360-Degree Protractor:** Measure physical or digital angles on-screen with precision.
2.  **Ring Sizer Chart:** Find your exact US/UK ring size using a physical ring placed directly on your display.
3.  **Creative Web Grid Overlay:** Essential grid layout utility for web design prototyping.
4.  **Instant Unit Converter:** Seamless conversion between metric and imperial dimensions.
5.  **Scale-Accurate Printable Rulers:** Generate precise, printable PDF templates for real-world measuring.

## 📏 How to Calibrate for 100% Accuracy

To make your **virtual ruler** measure real-world units perfectly:

1.  Click the **Calibrate** button.
2.  Choose your calibration helper object (e.g., standard Credit Card or Coin).
3.  Place the physical object against your screen and adjust the slider until the virtual boundary matches your physical object perfectly.
4.  Your online ruler is now fully calibrated!

## 🛠️ Local Setup & Installation

To run this project locally on your machine for development or custom builds:

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v16+) and `npm` installed.

### 1. Clone the Repository
```bash
git clone https://github.com/zeeuskhan/onlineruler.git
cd onlineruler
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser to see the live application.

### 4. Build for Production
```bash
npm run build
```
